## Props
| Prop      | Accepts   | Description                                                       | Required |
| --------- | --------- | ----------------------------------------------------------------- | -------- |
| text      | `string`  | The text content to display inside the gradient animation.        | Yes      |
| duration  | `number`  | Transition duration (in seconds) for the mask movement animation. | No       |
| automatic | `boolean` | (Reserved) Determines if animation should play automatically.     | No       |
| className | `string`  | Additional class names for custom styling of the SVG wrapper.     | No       |
