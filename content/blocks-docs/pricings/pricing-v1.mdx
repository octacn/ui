```

```

## Props
| Prop          | Type       | Default     | Description                                           |
| ------------- | ---------- | ----------- | ----------------------------------------------------- |
| `id`          | `string`   | —           | Unique identifier for the pricing card.               |
| `name`        | `string`   | —           | Name/title of the pricing plan.                       |
| `time`        | `string`   | —           | Delivery time (e.g., "2 days").                       |
| `prices`      | `string`   | —           | Price of the plan (e.g., "$50").                      |
| `discount`    | `string?`  | `undefined` | Optional discount text (e.g., "20% off").             |
| `features`    | `string[]` | —           | List of features included in the plan.                |
| `className`   | `string?`  | `undefined` | Custom class names to style the card.                 |
| `isPopular`   | `boolean?` | `false`     | Whether the plan is marked as popular (adds a badge). |
| `description` | `string?`  | `undefined` | Short description of the plan.                        |
