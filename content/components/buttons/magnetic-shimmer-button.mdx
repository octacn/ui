## Props
| Prop             | Type            | Default                 | Description                          | Required |
| ---------------- | --------------- | ----------------------- | ------------------------------------ | -------- |
| shimmer-color    | string          | oklch(75% 0.183 55.934) | Color of the shimmer effect.         | No       |
| shimmer-size     | string          | 0.2em                   | Size of the shimmer cut.             | No       |
| shimmer-duration | string          | 4s                      | Duration of the shimmer animation.   | No       |
| border-radius    | string          | 100px                   | Border radius of the button.         | No       |
| strength         | number          | 5                       | Intensity of the magnetic pull.      | No       |
| background       | string          | undefined               | Background color of the button.      | No       |
| class-name       | string          | undefined               | Extra class names for custom styles. | No       |
| children         | React.ReactNode | undefined               | Content inside the button.           | No       |
| required         | boolean         | false                   | Whether the button is required.      | No       |
