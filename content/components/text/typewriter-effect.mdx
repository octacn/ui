## Props
| Prop        | Accepts             | Description                                                     | Required |
| ----------- | ------------------- | --------------------------------------------------------------- | -------- |
| children    | `string`            | The text content to display with the typing animation.          | Yes      |
| className   | `string`            | Additional CSS classes for styling the wrapper element.         | No       |
| duration    | `number`            | Delay (in ms) between typing each character.                    | No       |
| delay       | `number`            | Delay (in ms) before starting the typing animation.             | No       |
| as          | `React.ElementType` | Element type to render (default is `div`).                      | No       |
| startOnView | `boolean`           | If `true`, animation starts only when the component is in view. | No       |
| ...props    | `MotionProps`       | Additional motion props from `motion/react`.                    | No       |
