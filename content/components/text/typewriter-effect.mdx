## Props

| Prop        | Type                | Default | Description                                                     | Required |
| ----------- | ------------------- | ------- | --------------------------------------------------------------- | -------- |
| children    | `string`            | -       | The text content to display with the typing animation.          | Yes      |
| className   | `string`            | -       | Additional CSS classes for styling the wrapper element.         | No       |
| duration    | `number`            | `100`   | Delay (in milliseconds) between typing each character.          | No       |
| delay       | `number`            | `0`     | Delay (in milliseconds) before starting the typing animation.   | No       |
| as          | `React.ElementType` | `"div"` | Element type to render (e.g., `"h1"`, `"p"`, `"span"`).        | No       |
| startOnView | `boolean`           | `false` | If `true`, animation starts only when the component is in view. | No       |
| repeat      | `boolean`           | `true`  | If `true`, the animation will repeat continuously.              | No       |
| repeatDelay | `number`            | `2000`  | Delay (in milliseconds) between animation cycles when repeating.| No       |
| ...props    | `MotionProps`       | -       | Additional motion props from `motion/react`.                    | No       |

## Examples

### Basic Usage

```tsx
<TypewriterEffect>
  Welcome to our website!
</TypewriterEffect>
```

### Custom Speed and Delay

```tsx
<TypewriterEffect duration={50} delay={1000}>
  Fast typing with initial delay
</TypewriterEffect>
```

### No Repeat (One-time Animation)

```tsx
<TypewriterEffect repeat={false}>
  This message appears only once
</TypewriterEffect>
```

### Custom Repeat Timing

```tsx
<TypewriterEffect repeatDelay={5000}>
  This repeats every 5 seconds
</TypewriterEffect>
```

### Start on Scroll into View

```tsx
<TypewriterEffect startOnView>
  Animation starts when scrolled into view
</TypewriterEffect>
```

### Different HTML Element

```tsx
<TypewriterEffect as="h1" className="text-4xl font-bold">
  Heading with typewriter effect
</TypewriterEffect>
```

### Combined Configuration

```tsx
<TypewriterEffect
  duration={75}
  delay={500}
  repeat={true}
  repeatDelay={3000}
  startOnView={true}
  as="h2"
  className="text-2xl text-blue-600"
>
  Fully customized typewriter effect
</TypewriterEffect>
```
